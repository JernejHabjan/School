\begin{itemize}
\item Feasibility study / študija izvedljivosti\item Pilot Case / prototip\item Comparative study / primerjalna študija
\item Literature survey / pregled literature
\item Formal Model / formalni model
\item Simulation / simulacija
\end{itemize}
