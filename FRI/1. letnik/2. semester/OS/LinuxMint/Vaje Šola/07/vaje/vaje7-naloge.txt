14.	Ustvarite naključno geslo s pomočjo generatorja psevdonaključnih števil. Geslo naj bo sestavljeno iz sedmih alfanumeričnih znakov.



15. Napišite skripto, ki izračuna največji skupni delitelj treh naravnih števil, ki jih podamo kot argumente skripti.
Primer klica: ./najvecjiDelitelj.sh 27 15 45 

Največji skupni delitelj dveh naravnih števil lahko izračunamo rekurzivno:
int gcd(a, b){
    if (b == 0){
       return a;
    }
    else {
	    return gcd(b, a%b);
    }
}

Velja:
gcd(a, b, c) = gcd(a, gcd(b, c)) = gcd(gcd(a, b), c) = gcd(gcd(a, c), b)
